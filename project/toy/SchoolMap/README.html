SchoolMap

这个是课程设计的题目(校园地图)，主要是考察最短寻径算法。 但看其他人在网上下了一个GUI界面的项目，因此就打算自己也做成图形界面的。 这个程序应该是我学习C++后第一次实际写项目。 使用wxwidget完成的，linux下运行有点小问题(主要是开始没考虑gtk在linux下有 些widget是不接收部分消息的), windows下应该没问题。

有两个程序一个是制作地图数据包的，一个是”导航”的A。 制作好的湖经地图包由于习惯不好已经丢失了。
