LFShare

一款局域网内并发传输软件使用web作为界面, 使用UDP进行广播传输在有n个接收者时能将局域网带宽"提高"近n倍。

aur

在AUR上维护的一些软件包的安装脚本(PKGBUILD)。这个不足以单独作为一个项目，只是放在toy里又不适合所以勉强独立出来。

hbue_thesis

湖北经济学院毕业论文latex模板文件，使用这个模板可以方便的使用latex来编写毕业论文而不用担心格式方面的问题。

AppWebServer

一款小巧的C++ httpserver, 内建jsonp实现主要目的是方面使用web作为GUI界面。适合用c++而又不想使用qt,wx,gtk等大型GUI库(程序尺寸问题)的同学。

WHMon

本软件是一款简单的教室管理软件，支持远程查看、远程操作、发送消息、批量关机、权限分组等。

JJJ

本工具可以辅助快速生成经销商需要制作的家电下乡销售报表。

toy

一些零碎的小项目，涉及到perl、python、ruby、opencv、network、hack等。
